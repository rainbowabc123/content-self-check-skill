## [1.0.0] - 2026-06-13

### Added
- Initial release of Content Self-Check Skill
- 10 comprehensive quality detection items:
  - Grammar detection
  - Logic gap detection
  - Numerical error detection
  - Citation error detection
  - Factual consistency detection
  - Formatting detection
  - Terminology detection
  - Emotional flow detection
  - Title-content matching detection
  - Information completeness detection
- Smart severity classification (Critical, Medium, Suggestions)
- Content type recognition (social media, articles, videos, etc.)
- Detailed reports with quoted text and revision suggestions
- Bilingual support (Chinese and English)
- MIT License for open-source use

### Features
- Zero dependencies - pure text skill definition
- Style-preserving error correction
- Cross-validation for consistency
- Auto-detection of content format
- Integration with Claude Code and compatible AI assistants
